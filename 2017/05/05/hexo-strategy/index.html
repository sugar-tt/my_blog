##搭建github pages +hexo for windows

##需要安装的程序

  • git
  • Node.js
  • npm
  • hexo

##安装git
从百度上下载后，执行exe文件进行安装。

##安装Node.js
根据自己的操作系统,从 http://nodejs.org/#download 下载。windows下载后直接执行node-v6.10.3-x64.msi,安装完成后将安装路径加到环境变量中。点击鼠标右键，执行git bash，验证安装成功。

##安装npm
根据自己的操作系统,从 https://github.com/isaacs/npm/tags 下载
